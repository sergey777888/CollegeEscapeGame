# EscapeCollege

## О проекте
**Escape College** — это атмосферный хоррор-квест, действие которого разворачивается в стенах заблокированного колледжа. Вы оказываетесь заперты в пустых коридорах учебного заведения, где каждый шаг может стать последним. Ваша единственная цель — выбраться.

## Основные механики
* **Исследование**: Изучайте локации, чтобы найти улики и предметы.
* **Головоломки**: Ищите спрятанные цифры и коды, чтобы открывать запертые двери.
* **Атмосфера**: Давящее чувство страха и неизвестности в замкнутом пространстве.

## Техническая информация
* **Движок**: Unity
* **Жанр**: Horror, Puzzle, Adventure

## Использованные ресурсы
* **Арт-контент**: Сгенерирован с использованием нейросетей (Midjourney/DALL-E) для создания давящей атмосферы хоррора.
* **Звуковое оформление**: Фоновая музыка и звуковые эффекты подобраны для усиления напряжения и чувства страха.

## Об авторе
Проект разработан в рамках олимпиады для демонстрации навыков разработки игровых механик, работы с компонентами Unity и базового геймдизайна.
